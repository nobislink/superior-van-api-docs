---
title: 'Get Lead'
openapi: 'GET /leads/{leadId}'
---

Retrieve the current status and notes for a submitted lead using the `leadId` returned from **Create Lead**.

<Note>Send `Authorization: Bearer <token>` with every request.</Note>

### Authorizations

<ParamField path="Authorization" type="string" required placeholder="Bearer <token>">
  Bearer access token issued to your vendor account.
</ParamField>

### Parameters

<ParamField path="leadId" type="string" required placeholder="lead_abc123">
  Lead ID returned from **Create Lead**.
</ParamField>

### Response fields

<ResponseField name="id" type="string">
  Lead identifier.
</ResponseField>
<ResponseField name="status" type="string">
  Current processing status (e.g., received, in_review, contacted, scheduled, closed, rejected).
</ResponseField>
<ResponseField name="notes" type="string">
  Internal notes about the lead.
</ResponseField>
<ResponseField name="createdAt" type="string" required>
  Lead creation timestamp (ISO 8601).
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://sandbox.superiorvan.com/v1/leads/lead_abc123" \
  -H "Authorization: Bearer <token>"
```

</RequestExample>

<ResponseExample>

```json 200 Response
{
  "id": "lead_abc123",
  "status": "received",
  "notes": "Lead received; assigned to Louisville team",
  "createdAt": "2025-01-12T18:05:00Z"
}
```

</ResponseExample>
